# 🎮 MazeEscape — 2D Maze Chase Game

## 📌 Overview

MazeEscape is a 2D maze-based chase game developed using **C++ and SDL2**. The game challenges players to navigate through a maze while avoiding obstacles and completing objectives.

This project demonstrates core concepts of **game development**, including rendering, movement logic, collision detection, and real-time user interaction.

---

## 🚀 Features

* 🎯 Maze-based gameplay
* 🕹️ Player movement and controls
* ⚡ Real-time rendering using SDL2
* 🧱 Collision detection system
* 🎮 Interactive game environment

---

## 🛠️ Tech Stack

* **Language:** C++
* **Graphics Library:** SDL2
* **Concepts Used:**

  * Game loops
  * Event handling
  * Collision detection
  * Object movement
